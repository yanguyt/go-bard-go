
# Go Bard Go

Repository create to consume easily bard api with Go

## Usage/Examples
It's simple as that, instaciate your Bard interface and get all the answers that you need.

```go
func(){

    bard := Bard("{Your bearer token}")
    bard.GetAnswers("This is real?")

}

```
